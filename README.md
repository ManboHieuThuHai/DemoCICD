# MyProject - Next.js Website

Website Next.js đơn giản để demo Git conflict resolution.

## Cài đặt

```bash
npm install
```

## Chạy development server

```bash
npm run dev
```

Mở [http://localhost:3000](http://localhost:3000) trong trình duyệt.

## Build cho production

```bash
npm run build
npm start
```

## Cấu trúc dự án

```
MyProject/
├── pages/           # Các trang Next.js
│   ├── index.js     # Trang chủ
│   ├── about.js     # Trang giới thiệu
│   ├── _app.js      # Root component
│   └── api/         # API routes
├── styles/          # CSS files
├── src/             # Source code cũ
└── package.json     # Dependencies
```

## Features

- ✅ Trang chủ với giao diện đẹp
- ✅ Trang giới thiệu
- ✅ API endpoint mẫu (/api/hello)
- ✅ Responsive design
- ✅ Gradient background

## Git Exercises

Dự án này được sử dụng để thực hành:
1. Git rebase conflicts
2. Large merge conflicts
3. Git stash scenarios
4. Git rerere automation
